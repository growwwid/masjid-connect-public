# masjid-connect-public
Landing Page untuk Website Masjid Connect. Website / Webapp ini akan kita gunakan untuk menghubungkan semua masjid dan musholla supaya bisa saling berinteraksi. Disini juga akan muncul beberapa informasi seperti Infomasi Kajian, Kebutuhan Antar Masjid, dan Informasi lainnya. Sehingga semua masjid dan musholla bisa saling terhubung.
